Common CSS Properties

Property

Property Value

Ruleset Example

Displays As

color

a color

p: {
     color: aqua;
}

Isn't this aqua color lovely?

font-family

type of font

p: {
     font-family: "fantasy", serif;
}

Fantasy isn't often used because it can be unpredicatably
displayed, but serif is quite common.

font-size

px - pixels
em - size relative to parent element
rem - size relative to root element

p: {
     font-size: 10 px;
}

This is what 10 px looks like.

font-weight

thickness of your font
ex:light, normal, bold

p: {
     font-weight: bold;
}

This is what bold text looks like.
