Common CSS Properties

Property

Property Value

Ruleset Example

Displays As

color

a color

p: {
     color: blueviolet;
}

Isn't this blueviolet color lovely?

font-family

type of font

p: {
     font-family: "Verdana", serif;
}

Here we have some text in a Verdana font.

font-size

px - pixels
em - size relative to parent element
rem - size relative to root element

p: {
     font-size: 10 px;
}

This is what 10 px looks like.

font-weight

thickness of your font
ex:light, normal, bold

p: {
     font-weight: bold;
}

This is what bold text looks like.
